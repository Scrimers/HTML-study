JavaScript란

정적인 요소만을 지닌 HTML에 사용자와 상호작용 가능한 동적인 요소를 추가하기 위해 사용한다.

JavaScript 이용방법

웹브라우저가 JavaScript를 인식할 수 있도록 HTML코드로 알려줘야 한다.
아래와 같은 태그를 body태그 안에 쓰면 된다. <body> <script></script> </body>

write에 대하여

웹페이지에 글씨를 출력할 때 사용 document.wirte(1+1); document.wirte('wow');

이벤트에 대하여

onclick = 클릭 시 이벤트 발생
<input type="button" value="asd" onclick="alert('onclick')">
onchange = 내용물 변경 시 이벤트 발생 (내용물 변경 후 해당 영역에서 벗어나야 작동)
<input type="text" onchange="alert('onchange')">
onkeydown = 내용물에 입력 시(키를 눌렀을때) 이벤트 발생
<input type="text" onkeydown="alert('onkeydown')">
