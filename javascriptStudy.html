JavaScript란

정적인 요소만을 지닌 HTML에 사용자와 상호작용 가능한 동적인 요소를 추가하기 위해 사용한다.

JavaScript 이용방법

웹브라우저가 JavaScript를 인식할 수 있도록 HTML코드로 알려줘야 한다.
아래와 같은 태그를 body태그 안에 쓰면 된다. <body> <script></script> </body>

write에 대하여

웹페이지에 글씨를 출력할 때 사용 document.write(1+1); document.write('wow');

이벤트에 대하여

onclick = 클릭 시 이벤트 발생
<input type="button" value="asd" onclick="alert('onclick')">
onchange = 내용물 변경 시 이벤트 발생 (내용물 변경 후 해당 영역에서 벗어나야 작동)
<input type="text" onchange="alert('onchange')">
onkeydown = 내용물에 입력 시(키를 눌렀을때) 이벤트 발생
<input type="text" onkeydown="alert('onkeydown')">

.length에 대하여

문자열의 크기를 알고싶을때 쓴다.
'안녕'.length;

toUpperCase에 대하여

소문자를 대문자로 치환하고 싶을때 쓴다.
'asd'.toUpperCase();

indexOf에 대하여

특정 문자열의 index를 알고 싶을때 쓴다.
'asd'.indexOf('d');

trim에 대하여

공백을 제거하고 싶을때 쓴다.
(문자열이 포함되지 않은 앞, 뒤 공백만을 제거한다.)
' asd d '.trim();

변수와 사용법

var 변수명 으로 사용한다. var는 variable의 줄임말.
아래와 같이 사용할 수 있다.
var asd = '자바스크립트'; alert('저는 지금 '+asd+'를 공부하고 있습니다.');

querySelector

대상을 선택자로 선택하여 제어할 수 있는 것.
아래와 같이 사용할 수 있다.
※querySelector의 경우 조회된 결과값의 가장 첫번째것만 선택 및 반영이 된다. //태그로 선택 시 document.querySelector('span').style.color="red"; //클래스로 선택 시 document.querySelector('.spanClass').style.color="red"; //id로 선택 시 document.querySelector('#spanId').style.color="red";

getElementById

id로 대상을 선택하는 것이다.
아래와 같이 사용한다. document.getElementById('spanId').style.color="green"; getElementById 는 querySelector('#아이디명') 과 동일하다.

== 과 === 의 차이

== : 값의 일치여부만 비교
=== : 값, 데이터 종류를 비교
var NData = 1; var SData = "1"; if(NData == SData) { document.write("값이 일치함 / ") }else { document.write("값이 일치하지 않음 / ") } if(NData === SData) { document.write("값과 데이터 타입 전부 일치함") }else { document.write("값과 데이터 타입 전부 일치하지 않음") }
